Scientific work for the master thesis

This application implements object detection and classification using models based on deep neural networks working on the client browser. It also includes visualization techniques for the analysis of object classification in images based on deep neural networks.

Experiment 1

This experiment focuses on the application of Class Activation Mapping (CAM) techniques to visualize the regions of interest in images that are classified by deep neural networks.

Grad-CAM Grad-CAM++ Layer-CAM

Experiment 2

In this experiment, object detection is performed using Faster-RCNN. The first page shows the results using a big Faster-RCNN model, while the second page uses the lightweight CocoSSD model, both running on the client browser.

Big Faster-RCNN CocoSSD

Spectrogram

This pages plot spectrograms from audios. The first ist built only using vanilla javascript, the second is built using the library EssentiaJs
