Grad-CAM Interpretation

Implementing Grad-CAM in TensorFlow.js for a model like COCO-SSD is not natively supported, mainly because object detection models like SSD do not expose internal feature maps and class logits in a way that's necessary for proper gradient-based visualization (like Grad-CAM or Grad-CAM++).
